# README #

This REPO contains the file needed to plot the graphs used in training pusher simulatin
### Where do I get more Information? ###

* Visiting [Gymnasium] (https://gymnasium.farama.org/)
* Checking Pusher Module [Pusher Site](https://gymnasium.farama.org/environments/mujoco/pusher/)

### How do I get set up? ###

* Create a virtual environment using the following command: 
 ```
    python -m venv my_local_environment
    source my_local_environment/bin/activate
    pip install -r requirements.txt
 ```
* Verify that file main.py has set file name properly, like: registro_pusher.csv
* execute the following: "main.py"
* That's for now :D
* This version already has the final weights included after training


### Supported python versions ###
3.12 and older.
